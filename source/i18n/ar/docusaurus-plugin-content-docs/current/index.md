# زيك SDK

> **236 بروتوكولاً. 1,576 مشغلاً. واجهة برمجة تطبيقات واحدة. مقفل الطور حتى 1.287 هرتز.**

<div id="zeq-kernel-copy-mount"></div>

Zeq عبارة عن نواة حسابية للرياضيات التوليدية: سطح API واحد يأخذ بيان المشكلة، ويربطه بالموتر المتري KO42، وينفذ مقابل معادلة HULYAS الرئيسية ضمن القاعدة الزمنية HulyaPulse 1.287 هرتز، ويعيد إيصال [ZeqProof](/learn/concepts/zeqproof) يمكن التحقق منه - إلى ميزانية خطأ بنسبة 0.1٪ بشكل عام النسبية، ميكانيكا الكم، الفيزياء الكلاسيكية، الإشارات الحيوية، الشبكات و234 عائلة بروتوكولية أخرى.

وتنقسم هذه الوثائق إلى أربعة مسارات. اختر الخيار الذي يطابق ما تحاول القيام به الآن.

## [تعلم](/learn/concepts/) — ما هو Zeq ولماذا

المسار المفاهيمي ابدأ هنا إذا لم يسبق لك رؤية Zeq من قبل. يغطي HulyaPulse بتردد 1.287 هرتز، وشبكة Zeqond الزمنية، وR(t)، وKO42، والمعادلة الرئيسية، وبروتوكول المعالج المكون من 7 خطوات، وكيف تجعل إيصالات ZeqProof كل نتيجة يمكن التحقق منها بواسطة طرف ثالث.

## [إنشاء](/build/quickstart/introduction) — احصل على النتيجة خلال خمس دقائق

المسار العملي. التشغيل السريع لـ cURL، وعملاء اللغة (JavaScript، وPython، وGo، وRust)، ووصفات المجال (التصوير الطبي، وفيزياء الألعاب، والروبوتات، والطقس، والأجهزة، والطب الشرعي). تنتهي كل وصفة برابط إلى البروتوكولات المحددة التي تستخدمها.

## [المرجع](/reference/protocols) — كل بروتوكول، كل مشغل، كل نقطة نهاية

السطح الشامل. **236 بروتوكولًا** تم تنظيمها في 39 عائلة، و**1,576 مشغلًا** عبر QM/NM/GR/CS/Awareness، وكتالوج نقطة نهاية API الكامل مع مواصفات OpenAPI. استخدم شريط البحث - هذا كثير جدًا للتمرير.

## [Operate](/operate/rate-limits) — تشغيل Zeq في الإنتاج

حدود المعدلات ومعالجة الأخطاء وخطافات الويب وكيفية قيام الجهات الخارجية بالتحقق من ZeqProofs الخاصة بك دون مفتاح API.

## اختبار القيادة لمدة 30 ثانية

```bash
curl -X POST https://www.zeq.dev/api/zeq/pulse \
  -H "Authorization: Bearer zeq_ak_demo_4ac31b46196a56e10631a4bda1b310b0a147a122" \
  -H "Content-Type: application/json" \
  -d '{"domain":"newton","operators":["NM19","KO42"],"params":{"m":1,"a":9.81}}'
```

سوف تستعيد كائن `ZeqState` يحتوي على النتيجة، وعلامة Zeqond التي تم إغلاقه فيها، ومرحلة HulyaPulse، وHMAC ZeqProof الذي يمكنك التحقق منه مقابل `/api/zeq/prove/verify` بدون مصادقة.

هذا هو النظام بأكمله في مكالمة واحدة.
