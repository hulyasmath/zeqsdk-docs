> # ZEQ Framework — SDK Documentation
>
> The ZEQ Framework SDK is the developer interface for the Physics-as-a-Service platform at [zeq.dev](https://zeq.dev). It provides structured access to over 1,576 computational operators, resonance-aware tooling, and multi-locale reference documentation for building applications that align with physical law.
>
> ## What is the ZEQ Framework?
>
> ZEQ is a unified computation framework built on two discovered constants: **HulyaPulse (1.287 Hz)** — the fundamental heartbeat frequency — and the **Zeqond (0.777 seconds)** — the base temporal unit and state-lock interval. Together they form a reproducible foundation for physics-aligned software.
>
> The SDK allows developers, researchers, and AI systems to access the ZEQ operator library, query the API, and integrate the framework into any application. This repository is the source of the documentation hosted at [zeq.dev/sdk](https://zeq.dev/sdk).
